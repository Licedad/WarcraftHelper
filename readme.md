# WarcraftHelper

#### 介绍

魔兽辅助插件，解除地图大小限制，宽屏支持，解锁FPS，自动保存录像

如果你的显示器大于1080p，产生了字体重叠问题，游戏内使用F7键可以刷新窗口，需要窗口化模式。

1.20e和1.24e魔兽建议打上d3d8to9补丁。

录像会自动保存在魔兽replay目录的WHReplay子目录下。

支持版本：1.20e、1.24e和1.27a

使用方法：直接放到魔兽目录下。
